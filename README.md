# Construction Field Management App

## Tech Stack
- React.js (Vite)
- Tailwind CSS
- React Router v6
- Lucide Icons

## Features
- Login page with mock authentication
- Project dashboard with project cards
- Search projects
- Daily Progress Report form
- Image upload with preview (max 3 images)
- Form validation
- Responsive UI (mobile / tablet / desktop)

## Installation

Clone the repository:

git clone https://github.com/yourusername/construction-dpr-app.git

Go to project folder:

cd construction-dpr-app

Install dependencies:

npm install

Run project:

npm run dev
